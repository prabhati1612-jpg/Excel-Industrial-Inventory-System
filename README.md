# 🏭 Automated Industrial Inventory System (2026)

This project is a professional-grade Inventory Management System built entirely in **Microsoft Excel**. It uses a relational data model to link Master Data with daily Transaction logs.

## 🚀 Key Features
- **Real-time Tracking:** Automated Stock-In/Out calculation using `SUMIFS`.
- **Dynamic Dashboard:** Auto-updating summary sheet with `UNIQUE` and `XLOOKUP` formulas.
- **Inventory Alerts:** Visual signals (Red/Green) for reorder levels.
- **Scalable Design:** Built with 2026 Dynamic Arrays to handle thousands of rows.
